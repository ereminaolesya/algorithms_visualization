## О проекте
Проект создан для визуального представления работы алгоритмов оптимизации и кластеризации.  
Проект является групповым, в данном репозитории представлена часть работы.

## Ant Colony Optimization
Реализация муравьиного алгоритма для поиска оптимального пути.

Функциональность:
- генерация графа / точек
- обновление феромонов
- поиск кратчайшего пути

## Clustering
Реализация алгоритмов кластеризации.

Функциональность:
- генерация точек
- разбиение на кластеры
- 3 различных алгоритима
